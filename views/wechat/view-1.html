概述

用途

  1. 友邻优课微信端主要为三个产品：
    1. 友邻优课服务号：吸引流量、购买转化；
    2. 友邻优课小程序：听课及社交基础功能，按照张小龙价值观，打造“用完即走”的产品；
    3. 友邻优课收款号（多个）：仅作为收款工具，收款至集团其他子公司银行账户；
  2. 2016年12月31日前，微信端全部前端页面将逐步改版，具体见下方“时间规划”；
  3. 微信端后台与友邻优课管理员后台结合，具体文档见Admin一章；
  4. 改版完成后，友邻优课服务号开发工作将以【需微信登录的运营活动技术支持】为主；

功能概述

友邻优课服务号主要分为如下几个模块：会员购买、优课商城、个人中心、今日免费、经销商。
  1. 会员购买：分为新手购买、会员续费、赠送好友三大部分；
  2. 优课商城：类似淘宝，用户任意选购商品（即课程），完成支付后对接给相应客服；
  3. 个人中心：会员购买后填写礼品卡，送给好友；
  4. 今日免费：免费节目、社群活动等；
  5. 经销商：单独生成链接。用于经销商及运营、财务人员查看购买及返现数据，不在公众号显示入口；

时间规划

  2. 12月5日下线合伙人功能并完成经销商功能（合伙人改版）；
  3. 12月10日前完成会员购买功能的改版；
  4. 12月20日前完成商品体系的改版；
  5. 12月31日完成后台数据的对接；
