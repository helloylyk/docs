今天模块

概述

对于学员用户，以“天”为单位，用卡片的形式显示每天更新的课程。非会员第一张卡片为引导购买的卡片，后面的卡片与会员一致。

卡片列表

  1. 第一个卡片为“今天”，向左侧滑动依次显示最近5天更新的内容；
  2. 最后一个卡片左滑切换到课程页。
  3. 双击底部Tab Bar时自动返回到第一个卡片。
  4. 下拉刷新，如有更新，显示为最近的一天更新。

卡片详情

卡片可上下滑动，每个卡片需要包括以下内容：

  1. 更新日期以及更新的专辑数量
  2. 当日学习时长（单位分钟），显示截至刷新时间已学的分钟数
  3. 课程列表：当前日期内所有更新的课程，以及课程的相关信息。单个课程需要包括以下信息：
    1. 课程封面：点击课程封面跳转到播放页。每日新闻讲解版在顶部，并且以大图展示。
    2. 课程名
    3. 老师、课程时长
    4. 专辑名：点击专辑名可跳转到对应的专辑详情页。
  1. 会员点击课程跳转到对应的播放页
  2. 非会员点击课程提示“成为友邻优课学员即可收听此课程” 立即入学 咨询阿树老师 取消。点击【立即入学】跳转到购买页，点击【咨询阿树老师】跳转到网易七鱼。

非会员卡片

非会员第一张卡片样式及文案如下图，点击图片播放视频。

搜索

点击首页或知识左上角的【搜索】按钮，跳转搜索页面。输入要搜索的内容，点击键盘的确认，则显示搜索结果。

    搜索结果中需要对用户输入的关键字进行“高亮”处理。
  1. 课程：显示含有关键字的课程，包括课程标题、老师、所属专辑、时长。点击后进入相应课程。

更新提醒

    当app有新版本时，在首页弹出更新提示。
  1. title：发现新版本
  2. content：后台设置的内容
  3. button1立即更新：点击【立即更新】：安卓在APP内进行下载更新；iOS跳转到appstore对应页面。
    安卓APP内下载显示下载进度，下载完成后自动打开安装包。
  4. button2取消：点击【取消】，如后台设置强制更新，则退出app；如非强制更新，则可正常使用app。

购买页

非会员点击【立即报名】跳转到购买页，购买页可选则微信报名和支付宝报名两种方式。付款成功后会员时间相应延长，返回app跳转到购买成功页面。

购买成功

购买成功后，显示购买成功页，提示用户添加阿树老师。页面显示后台分配的阿树老师（头像、微信号信息）。点击【加好友】，则提示“xxx，已复制到剪贴板，请到微信添加朋友页面粘贴并搜索”取消 打开微信。点击【打开微信】则调出微信。
