播放模块

概述

播放模块是app最为核心的模块，用户可以在播放模块完成播放课程、写心得&发布心得的过程。

当前播放

点击课程或右上角的播放按钮可跳转播放页。关闭重新打开app，点击右上角的播放按钮，则跳转播放页，播放保持上次退出时的进度，并处于暂停状态。

播放页

  1. 打开播放页时，如用户在非wifi环境下使用，则不加载内容和音频，提示“确定使用流量听课？不再提醒 确定 取消，点击【确定】则正常播放，点击取消则返回上级页面，点击不再提醒则不再弹出类似提示。
  2. 如用户已下载或缓存了该课程，则无流量提示。
  3. 打开播放页自动播放。如上次未听完该课程，则提示“上次播放到xx:xx，是否继续”继续 取消。继续则跳转到上次播放位置播放。取消则直接开始播放。

下载

点击【下载】，则提示“已加入下载队列”。下载成功后，图标变为已下载图标。提示“下载成功”。

正文部分

正文部分的最上方显示课程的标题、所在专辑、老师名称、封面。标题如较长则两行显示

音频切换

  1. 当一个课程下有多个音频时，显示选择音频部分。
  2. 打开课程时，默认播放第一个音频，按钮显示音频名称。
  3. 点击音频名称打开弹窗，可选择其他音频。已选中的音频为选中状态，如图。点击已选中的音频无反应，点击其他音频则关闭弹窗并切换至选中的音频。同时按钮显示为选中的音频名。
  4. 下载音频则将多音频同时下载下来，在下载管理中仍显示为1节课。

音频切换下方显示课程材料。

播放条

进度条左边为已播放的时间，右边为剩余的播放时间

音频切换

  1. 当一个课程下有多个音频时，显示音频切换按钮。
  2. 打开课程时，默认播放第一个音频，按钮显示音频名称。
  3. 点击音频名称打开弹窗，可选择其他音频。已选中的音频为选中状态，如图。点击已选中的音频无反应，点击其他音频则关闭弹窗并切换至选中的音频。同时按钮显示为选中的音频名。
  4. 下载音频则将多音频同时下载下来，在下载管理中仍显示为1节课。

音频加载中

  1. 音频播放因非人为因素（网络状况等）暂停播放时，播放按钮显示加载动画。
  2. 如15s内仍未成功加载，则提示alert“网络状况不佳，建议更换4G或者Wifi” 好的。

|　播放列表

显示跳转之前已加载的课程列表信息。

  1. 从下载管理-专辑打开播放页：播放列表显示专辑中已下载的
    播放列表名称：专辑名
  2. 从下载管理-课程打开播放页：按照下载管理-课程的顺序显示已下载的所有课程
    播放列表名称：下载列表
  3. 从首页进入播放页：按照时间倒序显示当天已下载的所有课程
    播放列表名称：更新的日期，如“2月14日”
  4. 从右上角的播放进入播放页：保持当前的播放列表。

播放方式

点击播放方式按钮，可切换播放方式。

  1. 顺序播放：按照播放列表顺序自动播放课程，最后一节播放完毕从头开始播放
  2. 单节循环：自动循环播放当前课程。
  3. 单节播放：播放完当前课程后自动停止播放，并提示toast“单节播放完毕”
  4. 播放方式在切换课程后不改变。

每次点开播放列表，正在播放的课程默认在可见区域的最上方。

|　前进、后退10秒

|　播放/暂停

  1. 可通过耳机控制播放暂停
  2. 打电话/接电话时自动暂停，接完电话自动继续播放
  3. 离线及在线状态可统计时长，并在播放页中显示

|　功能菜单

页面右下角的功能菜单里包含点赞、心得、调节字号、倍速播放功能，

1.点赞

点击喜欢，则课程的喜欢数+1，该课程添加到我赞过的-课程中。再次点击则取消点赞，点赞数-1。

2.心得

点击心得跳转到心得列表页面，显示该课程下的心得。

3.倍速播放

可选0.8、1、1.25、1.5倍速，点击切换倍速。默认倍速为1倍。

4.调节字号

调节字号可选小、中、大三种字号，点击切换。切换课程或重新打开app继承字号大小。首次使用app默认为小号字体。

控制中心

iOS控制中心包含如下部分

  1. 播放&暂停：同播放页播放暂停。
  2. 上一首&下一首：按照播放列表顺序切换，如为最后一集或第一集则循环切换。
  3. 图片封面：选用该课程对应的专辑封面。
  4. 课程名称：如“哲学与科学”。
  5. 老师：如“夏鹏”。
  6. 所在专辑：如“哲学的故事”。

写心得

  1. 写心得按钮拖动效果：长按写心得按钮，可将其拖动到不超过文本区域的任意位置。拖动结束后记录位置。
  2. 点击【写心得】：按钮弹出写心得，默认文字提示“随时记录你的心得吧”，点心得外的区域可隐藏写心得。
  3. 点击写心得上方的区域：隐藏写心得，提示“本次编辑已自动保存” 我知道了 不再提醒。点击不再提醒则不再出现此提示。
  4. 心得内容每有变动则保存到本地一次。心得发布则清除已保存的心得。

|　播放/暂停

同播放条的播放暂停。

|　发布

  • 如心得无文字内容，提示“写点什么再发布吧”
  • 点击【发布】，显示“正在发布”及加载动画，如发布失败，toast“发布失败，请检查你的网络设置”
  • 如发布成功，则跳转到该课程的心得列表。如是当日第一次发布，则提示toast“每日发布心得，优币+5”，该用户优币+5；如不是第一次发布，提示toast“发布成功”

|　上传图片

上传图片按钮上以角标显示当前已选择上传的图片数量，如未上传图片，则不显示角标。
　　点击上传图片，如未选择图片，则跳转到选择图片页面。如已选择了图片，跳转到图片预览页。

选择图片

  1. 选择图片后，点击完成，如共选择了超过9张图片（选择的+在预览图中的），则提示“最多选择9张图片”
  2. 如未超过九张，点击完成，则图片添加为待上传图片，并在预览页中显示
  3. 点击取消，则不添加任何图片

点击完成后跳转到图片预览页。

图片预览页

  1. 点击底部的图片按钮跳转到图片预览页，预览的大图限制高度和宽度（宽度为屏幕宽度，高度限制以设计稿为准），小图按照1:1裁剪。
  2. 左右可滑动大图，小图联动。
  3. 点击删除则将该图片从待上传中删除。点击完成或返回，返回到写心得页。
