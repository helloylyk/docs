今天模块

模块概述

　　对于学员用户，以“天”为单位，用卡片的形式显示每天更新的课程。非会员第一张卡片为引导购买的卡片，后面的卡片与会员一致。

卡片列表

  1. 第一个卡片为“今天”，向左侧滑动依次显示最近5天更新的内容；
  2. 最后一个卡片左滑切换到课程页。
  3. 双击底部Tab Bar时自动返回到第一个卡片。
  4. 下拉刷新，如有更新，显示为最近的一天更新。

卡片详情

　　每个卡片需要包括以下内容：

  1. 更新日期以及更新的专辑数量，如日期为今天，则显示为“今日”。
  2. 当日学习时长（单位分钟），显示截至刷新时间已学的分钟数
  3. 课程列表：当前日期内所有更新的课程，以及节目的相关信息。

　　单个节目需要包括以下信息：

  1. 节目封面：点击节目封面跳转到播放页。由于每日新闻是周一到周五每日更新的节目，故节目封面在最上方以大图显示。
  2. 节目名
  3. 老师、节目时长
  4. 专辑名：点击专辑名可跳转到对应的专辑详情页。

　　会员点击课程跳转到对应的播放页；非会员点击课程提示“成为友邻优课学员即可收听此课程” 立即入学 咨询阿树老师 取消。点击立即入学跳转到购买页，点击咨询阿树老师跳转到学习管家页面。

非会员卡片

　　非会员第一张卡片样式如下图，点击播放视频。视频播放完成或取消播放，即弹出提示框，文案：三个按钮：

  1. 立即报名：跳转到购买页。
  2. 咨询阿树老师：跳转到学习管家页面。
  3. 暂时不想学习：关闭弹窗。
