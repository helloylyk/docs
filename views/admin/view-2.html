管理员体系

概述

  1. 管理员体系主要用于管理员的管理，包括添加、删除、修改基本信息及权限、查看管理员列表及信息功能；

添加管理员

    添加管理员，需要以下内容（默认无任何权限）：
  1. 用户名
  2. 初始密码
  3. 真实姓名

删除管理员

  1. 删除管理员记录，在操作前进行确认提醒；

修改管理员

  1. 用户名不可修改；
  2. 姓名：修改用户的真实姓名
  3. 职务：从上述6种角色中选择
  4. 权限
    1. 友邻优课后台管理系统主要有如下6种角色：运营总监、运营主管（用户）、运营主管（内容）、用户运营、内容运营、财务。其中运营总监有所有权限，其他角色具体权限如下：
      体系 增加 删除 修改 查询
      专辑
      运营主管（内容）
      运营主管（内容）
      运营主管（内容）
      运营主管（内容）
      节目
      运营主管（内容）
      内容运营
      运营主管（内容）
      内容运营
      运营主管（内容）
      内容运营（不可修改发布状态）
      运营主管（内容）
      内容运营
      用户
      运营主管（用户）
      用户运营
      运营主管（用户）
      运营主管（用户）
      用户运营
      运营主管（用户）
      用户运营
      心得 / / /
      运营主管（用户）
      用户运营
      打卡 / / /
      运营主管（用户）
      用户运营
      教师 / / / /
      管理员 / / / /
    2. 修改用户信息时，默认有如上5中选项，选择后列表中开放相应的权限；
    3. 用户也可以在选定职务后自定义权限；
  5. 密码：直接修改，无需输入之前密码，主要使用场景为管理员忘记密码后由后台管理员重置。
  6. 头像：用户自己上传

查看管理员

  1. 头像
  2. 姓名
  3. 用户名
  4. 创建时间
  5. 上次登录时间
  6. 权限列表
  7. 职务
