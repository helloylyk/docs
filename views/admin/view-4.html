节目体系

概述

  1. 友邻优课所有音频内容均以“节目”的形式呈现；
  2. 内容运营人员在后台对节目进行编辑，运营主管（内容）负责发布节目；

添加节目

  1. 添加节目仅需填写标题、标签和教师，在专辑列表页面弹出modal填写即可;
  2. 提交后跳转到查看节目页面，修改编辑其他信息

修改节目

  1. 封面图片：尺寸比例16:9；大小480*270；
  2. 标题：最多15字；
  3. 基本信息
    1. 标题：节目名称，限制20个字；
    2. 上线时间：若已上线，则不可修改上线时间，定时上线或未上线状态时可以修改上线时间；(只有运营主管（内容）可以修改节目的上线状态);
    3. 标签：从节目标签库中选择，多选；
    4. 教师：从本专辑教师列表选择；
    5. 初始收听人数：数字；
    6. 初始喜欢人数：数字；
  4. 上传音频：若已有音频，则显示文件名及时长，并且可以播放/暂停，并修改/删除音频；若尚未上传音频，只显示长按钮“上传音频”
  5. 节目内容：编辑人员根据需求添加元素，并进行编辑，具体规则如下；

内容样式

  1. 内容编辑人员会根据提前定义好的样式进行内容编辑；
  2. 编辑人员点击添加按钮后提示选择要添加的元素，确认后在页面中加入相应的元素，编辑人员根据提示边写文字（支持输入html控制颜色、加粗等样式）；
  3. 文字颜色默认为白色；
  4. 对已有的《每日新闻》专辑和《剧读莎士比亚》专辑进行分析，归纳总结出如下通用样式：
    类型 子类型 样式 备注
    卡片 标题 左侧黄色竖线 /
    正文 普通文字 /
    图片 / 宽度建议480，点击后弹出素材中心，可以上传不可以删除，具体见下方原型 /
    列表 / 数字列表，列表中的项目需用户逐个添加 下方设置“添加项目”按钮
    金句 / border，1px，黄色，高度与输入内容高度相同 用于显示金句等

删除节目

  1. 在操作前进行确认提醒；进行归档操作，防止误删造成重大损失；

查看节目

  1. 除可以编辑修改的信息外，还有以下内容：
    1. 时长：音频时长
    2. 上线状态
    3. 喜欢人数=实际+初始
    4. 收听人数=实际+初始
  2. 写心得人数；
  3. 心得数量；
  4. 收听人数；

发布预览

  1. 在编辑时或者编辑后可以模仿手机App页面，以用户的角度进行预览；
  2. 要求模仿手机端，以封面图片为背景，标题为导航栏标题，正文为内容；
  3. 点击“在手机中预览”按钮，弹出二维码，编辑人员扫码后即可在手机中查看真实效果；

节目分析

  1. 热度分析：选择时间范围、教师，显示该教师在指定范围内每天课程的实际“收听人数”、“写心得人数”、“点赞人数”。；
  2. 标签分析：扇形图显示各标签的人数占比。手动选择“收听”、“写心得”、“点赞”
